This map shows the public school enrollement across Washington state, using the data set "Public Schools Characteristics 2022-2023" from NCES' EDGE program, which develops annually updated point locations. The dashboard updates as the map is moved around and the bar chart shows the distribution of enrollement. Hovering individual circles shows the name of the school, the enrollement, and what level the school is.
