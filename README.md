For this map, I chose a proportional map because it allowed for the data to be visualized with more detail. If i had chosen to use a chloropleth map, it would have perhaps nto allowed for as much detail or information. Since the proportion of each circle gives the viewer data, and the hover function allows the viewer to see specific details about what kind of a school it is, how many people are enrolled and so on. A chloropleth map might be more appropriate for standardized data like percentages, but since the data is in counts, the symbol and colors both directly relate to the data size, which allows the viewer to understand the data better. While one might argue that the overlapping circles can make this harder to read and it would be better to use a chloropleth map, the ability to compare the size of schools as they are within same area would actually not have been possible with a chloropleth map.
